# Palindromic Number Detector

## Overview
This application checks if a given number is palindromic. A palindromic number is one that remains the same when its digits are reversed, such as 121 or 12321.

## Features
- Input any integer number.
- Check whether the number is palindromic or not.
- Simple and intuitive user interface.

## Installation
1. Clone the repository: 
   ```bash
   git clone https://github.com/hakimov-vs/Palindromic-number-detector.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Palindromic-number-detector
   ```
3. Install the necessary dependencies (if any):
   ```bash
   npm install
   ```

## Usage
Run the application using:
```bash
npm start
```
